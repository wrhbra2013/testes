# 15 Melhores Exercicios de Programacao

Colecao de exercicios classificados por dificuldade para praticavar logica de programacao.

## Estrutura

Cada linguagem contem:

- **Basico (1-5)**: Variaveis, condicionais, loops
- **Intermediario (6-10)**: Arrays, strings, funcoes
- **Avancado (11-15)**: Recursao, algoritmos, estruturas de dados

13. Contador de frequencia
